How to Use (30 Seconds)

  1. Double-click anti_doze.exe
  2. Allow camera access when prompted
  3. Minimize the window → Open your lecture (Zoom, Google Meet, etc.)
  4. Anti-Doze runs in background → Wakes you up with LOUD ALARM if you doze off
  5. Lecture over? → Open the app window → Press 'Q' to quit
